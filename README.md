
# Feedback Intelligence System

Feedback Intelligence System is a tool designed to help businesses understand customer opinions quickly and effectively. By analyzing user reviews and feedback, it extracts meaningful insights that can guide decision-making and improve customer experience.

## Features

- **Sentiment Analysis:** Automatically classifies feedback as positive, negative or neutral.
- **Data Cleaning:** Handles raw feedback and prepares it for accurate analysis.
- **Insight Extraction:** Highlights common themes and key points from large volumes of reviews.
- **Reusable Model:** Easy to integrate with new datasets for continuous monitoring.


## How to Use

1. Clone the repository:  
```bash
git clone https://github.com/chandanaashree/feedback-intelligence-system.git
```
2. Install required packages:

```pip install -r requirements.txt```

3. Run the main application:

```python app.py```

4. Explore the notebooks for analysis and experimentation.
